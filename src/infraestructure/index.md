Infraestructura
===============


La infraestructura de desarrollo de LliureX prevista para la versión 14.06 es la siguiente:


Una vez preparado un paquete se sube al PPA correspondiente de la versión en la que se está trabajando. 
En el caso de la versión 14.06, el _PPA_ es el _Platinum_. Se puede encontrar aquí:

* [Platinum](https://launchpad.net/~llxdev/+archive/platinum)

Si se quiere añadir:

```bash
sudo add-apt-repository ppa:llxdev/platinum
```
